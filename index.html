Programming with HTML and CSS

One new thing I am learning is how to program with HTML and CSS.
HTML is a webpage building code language that focuses mainly on the structure of the website.
CSS controls more of the styling of the website like coloring and borders.

HTML Code Specifics

Tags <body> <h1> <p> <img> <br>
Tags are the structure of HTML. They are surrounded by <>and the end tags also have a backslash. The body tag contains all HTML tags. Everything within the body shows up on the webpage. This tag is the header tag and makes all of the large titles in pages. To make subheaders, simply change the number to 2 or higher. The p tag is the paragraph tag. Inside it is where all of the text content is placed. The image tag is slightly different because it doesn't have an end tag. Instead, the image link is placed within the tag. The br tag makes line breaks. Similar to the image tag, there is no end tag.

CSS Code Specifics

Inline CSS Borders Colors Text Alignment Margins and Padding Fonts
Inline CSS is putting CSS coding within your HTML tags for more specific visual changes. The way this is done is writing style="" within the opening tag. Each separate command in CSS in the same tag must be separated by a semi colon. In the quotation marks, the CSS is written. Borders are the rectangular area surrounding a tag and can be changed by writing border:px; and adding a number in front of px (pixel). The border can also be different colors and styles and must be written within the style quotations, tag, or file. Some examples are border-color:; Colors in CSS are changed by writing color:; and either writing which color or writing rgb() and putting three numbers that are separated by commas in the parenthesis. RGB stands for red, green, and blue, so the three numbers within the parenthesis decide how much of each color makes up the final color. Text is aligned by writing text-align:; and writing either left, right, or center after the colon. As implied by the title, this command will cause text to begin at either the right, left, or center depending on what is written. All content in HTML has three surrounding layers: padding, border, and margin. The padding is between the border and content while the margin is the area between the border and other content. To change these, one must write margin:; padding:; within a style tag or quotation. After the colon, one can change the size of the margin and cause the content to shift up, down, left and/or right. Fonts can be changed in many ways including size and family(type). Since this is also CSS, it must be included within the style quotations. To change the font size, one must write font-size:px; and put a number in front of px. To change the font family, one must write font-family:; and write the family after the colon.
